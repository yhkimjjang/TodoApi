# ToDo API Server
---

## 개요
해당 프로젝트 Todo API 서버로서 Todo CRUD 기능을 제공한다.

### 주요기능
----
1. Todo에 대한 CRUD 기능을 제공한다
2. Todo Caregory에 대한 CRUD 기능을 제공한다.
2. 사용자 로그인 기능을 제공한다
3. 사용자 그룹 대한 CRUD 기능을 제공한다.
